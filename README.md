# 🎬 Netflix Clone

A responsive Netflix landing page clone built using HTML, CSS, and JavaScript. This project recreates the Netflix homepage with a modern, responsive user interface for learning front-end web development.

> **Note:** This project is for educational purposes only and is not affiliated with Netflix.

## ✨ Features

- Responsive Design
- Hero Section
- Trending Movies Carousel
- FAQ Accordion
- Modern UI
- Mobile Friendly

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
